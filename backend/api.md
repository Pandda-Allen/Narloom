前后端接口文档

1. '/' GET
   前端回传作品信息，内容以workId为参数，返回该作品信息，不涉及用户权限验证操作
2. '/#/login' GET
   前端回传用户登录信息，包含邮箱地址和密码，返回用户登录状态及用户信息
3. '/#/search' GET
   前端回传work参数，需识别tag，作者，id，分别进行搜索并返回匹配的works集合
4. '/#/register' POST
   前端回传用户注册信息，包含用户名，邮箱，密码，返回用户注册状态及用户信息
5. '/#/my-works' GET
   前端回传用户ID，返回该用户所有作品信息
6. '/#/favorite' GET
   前端回传用户ID，返回该用户所有收藏的作品信息
