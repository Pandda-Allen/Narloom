## 用户接口 (User)

### 用户注册
- API: `POST /user/register`
- Frontend Request:
```json
{
  "email": "user@example.com",        // 必填，邮箱格式
  "password": "password123",          // 必填，至少 6 个字符
  "name": "用户名",                    // 可选，默认为空
  "bio": "个人简介"                    // 可选，默认为空
}
```
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "Registration successful",
  "data": {
    "user_id": "uuid-string",
    "email": "user@example.com",
    "name": "用户名",
    "bio": "个人简介",
    "created_at": "2026-03-17 12:00:00"
  }
}
```
- Error Responses:
  - 400: Email and password are required / Invalid email format / Password must be at least 6 characters
  - 409: Email already registered

---

### 更新用户资料
- API: `PUT /user/<user_id>`
- Frontend Request:
```json
{
  "name": "新用户名",                  // 可选
  "bio": "新的个人简介",                // 可选
  "email": "new@example.com"          // 可选
}
```
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "User profile updated successfully",
  "data": {
    "user_id": "uuid-string",
    "email": "user@example.com",
    "name": "新用户名",
    "bio": "新的个人简介",
    "created_at": "2026-03-17 12:00:00"
  }
}
```
- Error Responses:
  - 400: Request body must be JSON / No fields to update
  - 404: User not found

---

### 删除用户
- API: `DELETE /user/<user_id>`
- Frontend Request: 无（user_id 从 URL 路径获取）
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "User and all related data deleted successfully",
  "data": {
    "user_id": "uuid-string",
    "deleted_assets": 5,
    "deleted_works": 2
  },
  "count": 1
}
```
- Error Responses:
  - 404: User not found
  - 500: Failed to delete user

**注意**：删除用户会级联删除以下数据：
1. 用户关联的所有 assets（包括 OSS 中的图片和 MongoDB 中的 asset_data）
2. 用户关联的所有 works（包括级联删除所有章节）
3. 用户记录本身

## 资产接口 (Asset)

### 创建新资产
- API: `POST /rest/v1/asset/createNewAsset`
- Frontend Request:
```json
{
  "user_id": "uuid-string",           // 必填
  "type": "character",                // 必填，enum: character|world|comic
  "work_id": "uuid-string",           // 可选，关联的作品 ID
  "asset_data": {                     // 可选，MongoDB 存储的详细数据
    // character 类型示例:
    "name": "角色名称",
    "description": "角色描述",
    "role": "主角",
    "traits": {"trait1": "勇敢", "trait2": "聪明"},
    "factions": "阵营名称",
    "group": "分组名称",
    "relationships": {"name": "关系描述"},

    // world 类型示例:
    "name": "地点名称",
    "description": "地点描述",
    "location": "地理位置",

    // comic 类型示例（由系统自动填充）:
    "type": "comic_picture",
    "oss_url": "https://oss-url.com/xxx.jpg",
    "oss_object_key": "comic/user123/2026/03/xxx.jpg",
    "original_filename": "image.jpg",
    "file_size": 102400
  }
}
```
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "Asset created successfully",
  "data": {
    "asset_id": "uuid-string",
    "user_id": "uuid-string",
    "work_id": "uuid-string",
    "asset_type": "character",
    "created_at": "2026-03-17 12:00:00",
    "updated_at": "2026-03-17 12:00:00",
    "asset_data": {}
  },
  "count": 1
}
```
- Error Responses:
  - 400: Missing required fields (type, user_id)
  - 500: Failed to create asset

```json
      "created_at": "2026-03-17 12:00:00",
      "updated_at": "2026-03-17 12:00:00",
      "asset_data": {}
    }
  ],
  "count": 1
}
```
- Error Responses:
  - 400: Missing required fields (user_id) / limit and offset must be integers

```json
{
  "asset_id": "uuid-string"           // 必填
}
```
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "Asset deleted successfully",
  "data": null,
  "count": 1
}
```
- Error Responses:
  - 400: Missing required fields (asset_id)
  - 404: Asset not found or delete failed

**注意**：删除资产会级联删除：
1. MongoDB 中的 `asset_data`（存储详细数据）
2. 如果是 `comic` 类型资产，会同时删除阿里云 OSS 中存储的图片文件

## AI 接口 (AI)

### 处理 AI 请求
- API: `POST /rest/v1/ai/process`
- Frontend Request:
```json
{
  "user_id": "uuid-string",           // 可选
  "task_name": "任务名称",             // 可选
  "task_type": "chat",                // 必填，enum: enhance|abstract|generate|chat|translate|summarize|rewrite|code|analysis
  "model": "deepseek-chat",           // 可选，默认 deepseek-chat
  "content": {
    "user_prompt": "用户提示词",        // 必填
    "system_prompt": "系统提示词"       // 可选
  },
  "parameters": {
    "temperature": 0.7,               // 可选，默认 0.7
    "max_tokens": 500                 // 可选，默认 500
  },
  "constraints": {
    "language": "zh-CN",              // 可选
    "tone": "专业但有趣"               // 可选
  }
}
```
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "AI request processed successfully",
  "data": {
    "task_id": "task-uuid",
    "result": "AI 生成的内容",
    "usage": {
      "prompt_tokens": 100,
      "completion_tokens": 200,
      "total_tokens": 300
    }
  },
  "endpoint": "/ai/process",
  "request_timestamp": "2026-03-17T12:00:00"
}
```
- Error Responses:
  - 503: AI Service not initialized

---

### 获取支持的模型列表
- API: `GET /rest/v1/ai/models`
- Frontend Request: 无
- Backend Response (Success):
```json
{
```

### 生成动画（图生视频）
- API: `POST /rest/v1/anime-tool/generateAnime`
- Content-Type: `application/json` 或 `multipart/form-data`
- Frontend Request (JSON 方式 - 使用已有图片):
```json
{
  "user_id": "uuid-string",             // 必填，用户 ID
  "asset_id": "uuid-string",            // 可选，已上传图片的资产 ID
  "oss_object_key": "xxx.jpg",          // 可选，OSS 对象键（与 asset_id 二选一）
  "parameters": {                       // 可选，生成参数
    "duration": 5,                      // 视频时长（秒）
    "fps": 24,                          // 帧率
    "style": "anime",                   // 风格类型
    "motion_strength": 0.7,             // 运动强度
    "prompt": "微风吹过头发"              // 生成提示词
  }
}
```
- Frontend Request (Form 方式 - 上传新图片):
```
user_id: uuid-string                    // 必填
picture: <file>                         // 可选，新图片文件
parameters: '{"duration": 5, "fps": 24}' // 可选，生成参数 JSON 字符串
```
- Backend Response (Success):
```json
{
  "status": "success",
  "message": "Anime generation task created (placeholder - implementation pending)",
  "data": {
    "task_id": "uuid-string",
    "status": "pending",
    "source": {
      "type": "asset",                  // asset | oss | new_upload
      "asset_id": "uuid-string",
      "object_key": "comic/user123/2026/03/xxx.jpg",
      "url": "https://oss-url.com/xxx.jpg"
    },
    "parameters": {
      "duration": 5,
      "fps": 24
    },
    "note": "This is a placeholder response. Video generation logic to be implemented."
  },
  "count": 1
}
```
- Error Responses:
  - 400: Request body must be JSON or multipart/form-data / Missing required fields (user_id) / Must provide either asset_id, oss_object_key, or picture file
  - 403: Unauthorized: This asset does not belong to the user
  - 404: Asset not found
  - 500: Failed to upload picture / Failed to get picture URL

## 状态码说明
| 状态码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | 认证失败 |
| 403 | 权限不足 |
| 404 | 资源未找到 |
| 409 | 资源冲突 (如邮箱已注册) |
| 500 | 服务器内部错误 |
| 503 | 服务不可用 |
